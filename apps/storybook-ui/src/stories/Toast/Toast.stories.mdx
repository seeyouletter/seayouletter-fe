# Toast

토스트는 기본적으로 사용자에게 메시지를 띄우는 데 사용합니다.  
특히 사용자의 화면을 제어하는 모달과 달리, 사용자의 인터페이스를 방해하지 않으면서 메시지를 띄우기 용이한 장점이 존재합니다.
기본적으로 직관적인 내용을 전달하기 위해 한 줄을 지향합니다.  
하지만 때에 따라서 부가적인 설명이 필요할 때에는 짧게 두 줄로 보이도록 합니다.
주의할 점은, 토스트는 사라지는 시간이 존재합니다.  
따라서 너무 많은 글을 전달해야 한다면 모달을 사용하도록 합니다.

export const TemplateToast = (args) => <ToastBox {...args} />;
export const TemplateSuccess = (args) => <ToastBox type="success" {...args} />;
export const TemplateError = (args) => <ToastBox type="error" {...args} />;
export const TemplateWarning = (args) => <ToastBox type="warning" {...args} />;
