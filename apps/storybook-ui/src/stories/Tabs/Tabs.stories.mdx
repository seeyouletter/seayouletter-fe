# Tabs

여러 개의 Tab들을 `TabsGroup`에서 받으면, Tab을 생성해요!  
카테고리에 따른 분류를 선택할 때 사용하기 위한 용도입니다.

<Meta
  title="Components/Tabs"
  component={TabGroup}
  argTypes={{
    tabs: {
      name: 'tabs',
      description: '렌더링할 탭에 관한 정보들입니다.',
      control: {
        type: 'array',
      },
      table: {
        type: {
          summary: 'TabInterface[]',
          detail: `
{
  id: string;
  label: string | React.ReactNode;
  onClick: () => void;
}
          `,
        },
        defaultValue: {
          summary: '[]',
        },
      },
    },
    tabWidth: {
      name: 'tabWidth',
      description: '탭의 너비를 받습니다.',
      control: {
        type: 'text',
      },
      table: {
        type: {
          summary: 'string',
        },
        defaultValue: {
          summary: '120px',
        },
      },
    },
    tabHeight: {
      name: 'tabHeight',
      description: '탭의 높이를 받습니다.',

        type: 'text',
      },

        type: {
          summary: 'string',

        },
      },
    },
  }}
/>

## `tabs`의 `label`은 자유입니다.

만약 `label`을 `HOC` 방식으로 전달하고 싶다면, 가능하도록 지원했어요! 🙆🏻
